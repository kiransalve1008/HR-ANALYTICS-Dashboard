# HR-ANALYTICS-Dashboard
This HR Analytics dashboard is built using Power BI to analyze employee data and identify key factors influencing attrition, workforce demographics, salary distribution, and experience levels. The dashboard helps HR teams make data-driven decisions to improve employee retention and workforce planning.

**HR Analytics Power BI Dashboard**
**Project Overview**
This HR Analytics dashboard is built using Power BI to analyze employee data and identify key factors influencing attrition, workforce demographics, salary distribution, and experience levels.
The dashboard helps HR teams make data-driven decisions to improve employee retention and workforce planning.

•	Provide actionable insights for workforce planning

Step-by-Step Process Used

•	Imported HR employee dataset into Power BI Desktop
•	Data included employee demographics, salary, job role, education, and attrition status
Data Cleaning & Transformation (Power Query)
•	Removed unnecessary and sensitive columns
•	Handled missing values
•	Created age groups and salary slabs
•	Ensured correct data types for numeric and categorical fields
Data Modeling
•	Designed a clean data model for HR analysis
•	Optimized tables for performance and clarity

•	Total Attrition

•	Average Age
•	Average Salary
•	Average Years at Company
Dashboard Design
•	Used KPI cards, bar charts, column charts, and slicers
•	Applied HR-friendly color theme
•	Focused on clarity and business readability
________________________________________
Key Insights & Analysis
•	High Attrition Rate (20.6%) indicates retention challenges
•	Employees aged 26–35 show the highest attrition
•	Attrition is higher among lower salary slabs (up to 10K)
•	Sales Executive & Sales Representative roles experience maximum attrition
•	Employees with 0–2 years at the company are more likely to leave
•	Male attrition slightly higher than female
